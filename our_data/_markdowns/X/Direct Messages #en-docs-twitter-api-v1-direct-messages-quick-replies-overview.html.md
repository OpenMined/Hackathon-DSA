Overview

Messages published with POST direct\_messages/events/new (message\_create) can have an attached Quick Reply to request stuctured input from a user.

Quick Reply Options API referenceList of up to 20 predefined options presented for a user to choose from.

Quick Reply attachment metadata will not be accessible via legacy REST and Streaming endpoints. Only text values will be delivered in the legacy Direct Message JSON format.
